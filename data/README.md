# Data

Scraped / collected datasets used by Hermes Research Station.

## `cas_oa_journals.json`

中科院 (Chinese Academy of Sciences) 开放获取 (OA) 期刊目录。

- **来源**: DOAJ (Directory of Open Access Journals) API
- **采集日期**: 2026-05-19
- **总数**: 123 本期刊
- **结构**: DOAJ API 原生 JSON 格式，每本包含 bibjson 元数据、审稿流程、创建日期等
- **用途**: 用于学术文献检索测试、OA 期刊分析、化学/材料/药学相关筛选（其中 13 本与化学/材料/药学相关）
- **发现**: 中科院分区表自 2026 年起停止更新
